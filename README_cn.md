# Appdowngrade

## 功能
一款应用降级工具。
1. 登录账号
2. 输入连接查询历史版本，下载
3. 安装、分享、删除应用

## 用法
巨魔或签名使用。登录-查询-下载-安装。

## 其他
### 项目基于 
1. [MuffinStoreJailed-Public](https://github.com/mineek/MuffinStoreJailed-Public)
2. [ipatool.js](https://github.com/wf021325/ipatool.js)
项目基于上面两个项目修改而成，修改了很多在使用 MuffinStoreJailed-Public 闪退的问题。目前本人使用正常。
